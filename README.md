# Spidey Montessori

Spidey Montessori is a free, local-first learning app for a parent and young child to use together for about 15 minutes a day.

The app uses a cheerful "web hero" theme with red, blue, stars, badges, and mission language. It avoids official Spider-Man, Marvel, Disney, character, logo, or copyrighted assets.

## MVP

The first version includes:

- Letter sound activity
- Number counting activity
- Visual addition activity
- 4-card memory matching activity
- Browser speech prompts
- Effort-based star rewards
- Parent-controlled reward redemption
- Milestone tracker
- Browser `localStorage` persistence

## Learning Goals

The app is designed for parent-guided practice in:

- Alphabet recognition
- Letter sounds
- Number recognition
- Counting
- Early addition with visual objects
- Memory practice
- Progress tracking
- Positive effort-based reinforcement

## Daily Use

A typical session is organized as:

- 3 minutes letters
- 3 minutes numbers
- 3 minutes early math
- 3 minutes memory
- 3 minutes celebration and review

Parents can repeat activities, manually award effort stars, redeem stars for real-life rewards, and update milestone progress.

## Technical Notes

This is a static web app built with plain HTML, CSS, and JavaScript.

- No backend
- No login
- No paid APIs
- No external assets required
- Progress is stored in the browser with `localStorage`
- Hostable on GitHub Pages

## Local Development

Open `index.html` directly in a browser, or serve the folder with any static file server.
