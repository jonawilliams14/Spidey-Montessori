# Project: Spidey Montessori

Build a free, local-first web app for a parent and two-year-old child to use together for about 15 minutes per day.

## Goal

Create a playful, parent-guided learning app focused on alphabet mastery, letter sounds, number recognition, counting, early addition and subtraction, memory improvement, milestone tracking, and effort-based rewards.

The child already knows most letter sounds and has practiced in workbooks, so the MVP can begin slightly above zero.

## Important Theme Note

The child loves Spider-Man and *Spidey and His Amazing Friends*.

Use a "web hero" / superhero-inspired theme with red, blue, webs, stars, badges, and friendly mission language.

Avoid official Spider-Man, Disney, Marvel names, logos, characters, images, or copyrighted assets unless the parent supplies them for private use.

## Core Product Principles

- Parent and child use the app together.
- Daily use should take about 15 minutes.
- The app should be interactive for the child.
- The app should use voice/audio prompts.
- The child should answer by tapping, matching, choosing, or counting.
- Keep the UI cheerful but calm.
- Avoid ads, videos, external links, timers, pressure, or overstimulation.
- Reward effort, completion, repetition, and improvement, not only correct answers.

## First MVP Scope

Build only:

- one letter sound activity
- one number counting activity
- one visual addition activity
- one 4-card memory activity
- star count
- reward store
- milestone tracker
- localStorage persistence
- audio prompt button

Do not build accounts, backend, user profiles, advanced curriculum generation, official Spider-Man assets, or complex settings in the first version.

## Success Criteria

A parent can open the app, start a learning activity in under 30 seconds, complete a 15-minute session with the child, award stars, redeem stars for real-life rewards, and update milestone progress.

The child can interact by tapping large buttons, matching cards, counting objects, and choosing answers with parent help.
